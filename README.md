## xesim简介

Xesim，让国行手机无损秒支持 eSIM，全球 eSIM 套餐任选；一卡多号，工作事务与生活隐私分开；支持多品牌多型号，iOS 也可下载 eSIM；搭配 Xesim App，轻松下载并管理 eSIM。

Xesim 与此前的 5ber eSIM 实现类似功能：利用写卡器将 eSIM 卡号复写到实体 SIM 卡中，帮助不支持 eSIM 卡的智能手机通过 Xesim 使用并切换不同的 eSIM 手机卡。
